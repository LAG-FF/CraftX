Deployment Instructions:

1. Create an assets folder in your project directory

2. Place your images in the assets folder

3. Create a package.json file with the following content:

{
  "name": "craftx-website",
  "version": "1.0.0",
  "scripts": {
    "vercel-build": "echo 'No build step needed for static site'"
  },
  "devDependencies": {}
}

4. Deploy to Vercel using their GitHub integration or CLI
